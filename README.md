# 📱 Sistema de Control de Asistencia Móvil

![Flutter](https://img.shields.io/badge/Flutter-02569B?style=for-the-badge&logo=flutter&logoColor=white)
![Firebase](https://img.shields.io/badge/Firebase-039BE5?style=for-the-badge&logo=Firebase&logoColor=white)

Aplicación móvil desarrollada en **Flutter** para el registro de asistencia laboral mediante **validación GPS, evidencia fotográfica (registro de presencia) y modalidad remota**. Proyecto final de curso.

---

## ✨ Características principales

- 🔐 **Inicio de sesión seguro** con Firebase Authentication (correo institucional).
- 📍 **Validación de geolocalización** con perímetro configurable desde Firestore.
- 📸 **Evidencia fotográfica de presencia**: Captura de imagen como respaldo visual.
- 🏠 **Modalidad Remota**: Permite registrar asistencia desde casa sin validación GPS.
- 📋 **Historial de asistencia** con distinción visual entre "Presencial" y "Remoto".
- 📄 **Exportación de reportes en PDF** directamente desde la app.
- 📊 **Exportación a Excel (CSV)** para análisis de datos en hojas de cálculo.
- 🌙 **Modo Oscuro** integrado con persistencia de preferencia.
- 🎨 **Diseño premium** con Glassmorphism, degradados y animaciones suaves.
- 👤 **Perfil de usuario** con edición de datos personales y cambio de contraseña.

---

## 🛠️ Tecnologías utilizadas

| Tecnología | Uso |
| :--- | :--- |
| **Flutter / Dart** | Desarrollo de interfaz y lógica móvil |
| **Firebase Authentication** | Inicio de sesión y gestión de usuarios |
| **Cloud Firestore** | Base de datos en tiempo real para registros de asistencia |
| **Firebase Storage** | Almacenamiento seguro de evidencia fotográfica |
| **Geolocator** | Validación de ubicación GPS |
| **Camera** | Captura de evidencia visual con cámara frontal |
| **PDF & Printing** | Generación y exportación de reportes |
| **CSV** | Exportación de datos a formato Excel |
| **Provider & Shared Preferences** | Gestión de tema oscuro y preferencias |

---

## 🚀 Flujo de uso

1. Inicia sesión con tu correo institucional.
2. Activa la **Modalidad Remota** si estás trabajando desde casa.
3. Presiona **"Registrar Entrada"** o **"Registrar Salida"**.
4. Acepta los permisos de GPS y Cámara.
5. Captura tu **evidencia fotográfica** para confirmar tu asistencia.
6. Revisa tu historial y descarga tu reporte en **PDF o Excel (CSV)**.
